# 🏥 Banco de Dados - Clínica

Projeto de banco de dados desenvolvido para simular o funcionamento de uma clínica médica.

## 📌 Funcionalidades

* Cadastro de médicos, pacientes e funcionários
* Controle de consultas
* Relacionamento entre tabelas
* Consultas SQL com JOIN, UNION e subqueries

## 🗂 Estrutura do Banco

* **Ambulatorio**: salas e capacidade
* **Medico**: dados dos médicos
* **Paciente**: dados dos pacientes
* **Funcionario**: equipe da clínica
* **Consulta**: agendamentos

## ⚙️ Tecnologias

* MySQL

## ▶️ Como executar

1. Criar o banco de dados
2. Executar os scripts na ordem:

   * create_tables
   * inserts
   * queries

## 💡 Observações

Projeto com fins educacionais para prática de SQL e modelagem de dados.
